                用了两天时间把《天道》给看完了，昨晚（20220307）通宵看到了5点。 目前为止，通宵看完的电视剧有两部，一部是《我是特种兵》，在西屋看的光盘。一部是《天道》，在书房看的乐视视频。

                之所以能通宵看完，有以下客观原因：

                1. 剧情好看
                2. 集数较少
                  • 《特种兵》25集
                  • 《天道》24集

                看书、看电视、亦或是看电影。提高效率的方法只有一个：写笔记/写思考。这既是与自己对话，也是提炼重点。

                # 股票

                丁元英无论是私募，还是与欧阳的对赌，期限都是近一年。就这一条，就屠杀了多少要期望暴富/挣快钱的弱势文化的群体。

                这个问题很复杂，有技术面、制度面、产业结构……很多因素，我跟你说不明白。这东西有点像禅，知之为不知，不知更非知。

                # 自立 但不孤立

                即使已经靠卖收藏的唱片度日，在年关接到小丹的电话的时候，依然是：有事么？

                这说明靠自己解决问题的思想已经是他的本能。

                但是他并不是把自我封闭起来，当他需要他人的帮助而成事的时候，也会给好友韩楚风打电话。

                在道的方面，他是强势的、自立的；在术的方面，他又是客观的，深谙造势/借力。

                # 全力投入

                先不论丁元英的人脉、格局、能力。就是他做事的态度，都值得自己学习。

                既然打算要搞音响，就身心投入：查资料、做分析、实地走访、住在村里。

                这做事的劲头，不需要刻意为之，已经是他的习惯了。

                # 奢简自如，爱好相随

                无论是在没有空调的顶楼，还是在各种高档场所。一律能平淡处之...西餐也能吃，路边小摊也能吃，泡面也能吃。

                说明他的适应环境的能力很强，或者说他不在乎这些物质上的。

                但同时无论他在何种环境下，音响、喝茶、抽烟，这些他的爱好都伴随着他。

                # 强势文化

                透视社会依次有三个层面， 技术、 制度、和文化。小到一个人，大到一个国家一个民族任何一种命运归根到底都是那种文化属性的产物。强势文化造就强者， 弱势文化造就弱者，这是规律。也可以理解为天道，不以人得意志为转移。

                强势文化就是遵循事物规律的文化，弱势文化就是依赖强者得道的期望破格获取的文化，也是期望救主的文化。强势文化在武学上被称为秘笈， 而弱势文化由于易学、 易懂、易用、成了流行品种。

                强盗的本质是破格获取，破格获取和直接获取是两个不同的概念。你们没有自信与强者在同一个规则下竞争，这只能说明你是弱者，因为弱势文化所追求的最高价值就是破格获取。所以，强盗的逻辑从本质上讲是懦弱的生存哲学，所以你不算好汉。

                是皇天在上的文化，是救主、救恩的文化。如果一个民族的文化从骨子里就是弱势文化属性，怎么可能去承载强势文化的政治、经济？衡量一种文化属性不是看它积淀的时间长短，而是看它与客观规律的距离远近。五千年的文化是光辉、是灿烂，这个没有问题。但是，传统和习俗得过过客观规律的筛子。

                我们这个民族总是以有文化自居，却忘了问一句，是有什么文化，是真理真相的文化，还是弱势文化；是符合事物规律的文化，还是违背事物规律的文化，归根到底都是那种文化属性的产物，不以人的意志为转移。

                中国的传统文化是皇恩浩荡的文化，他的使用是以皇天在上为先决条件。中国为什么穷，穷就穷在幼稚的思维，穷在期望救主，期望救恩的文化上，这是一个渗透到民族骨子里的价值判断体系。太可怕了！

                马克思主义得道理归根到底一句话，客观规律不以人的意志为转移，什么是客观规律，归根到底也是一句话：一切以时间，地点和条件为转移。

                丁元英是一个破解了事物运行和发展规律密码的人，凡人是用人性来处理事情，而丁元英是借用事物的运行规律来看待事情。

                传统观念的死结就在一个“靠”字上，在家靠父母，出门靠朋友，靠上帝，靠菩萨，靠皇恩，总之靠什么都行，就是别靠自己。

                咱们翻开历史看看，你从哪一行哪一页能找到救世主救世的记录？没有，从来就没有，从来都是救人的被救了，被救的救了人。如果一定要讲救世主的话，那么符合和代表客观规律的文化就是救世主。

                强势文化的特点：

                1. 心胜
                2. 靠自己：不期望/不相信救主
                3. 知规律，顺规律
                4. 没有万能：道就是规矩，既有所能就必有所不能。
                弱势文化：等靠要
强盗逻辑：“破格获取”，为达到自己的目的破坏既有规则规律

强势文化就是以实事求是的态度，去遵循客观规律的文化，万事皆按规律办事，不以人的意志为转移的实事求是。
弱势文化就是依赖强者的道德，期望破格获取、不劳而获，梦想一本万利、一夜暴富，期待救主的文化，本质上就是依赖和跟随。

                # 缠中说缠

                生存法则就是忍人所不忍，能人所不能。忍是一条线，能是一条线，两者之间就是生存空间！简言之，讲的是人的能耐问题。能是前锋攻击力，耐是后卫防守力。
