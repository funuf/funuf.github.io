# 概述

经济就像一部简单的机器那样运行，但很多人不懂得这点。

经济虽然可能看起来复杂，但是其实是以简单且机械的方式运行。

经济由几个简单的零部件，和无数个重复的简单交易组成。

  • 生产率的提高
  • 短期债务周期
  • 长期债务周期

# 交易

在每次交易中，买方使用货币或信用，向卖方交换商品/服务/金融资产。

货币+信用=支出总额 支出总额是经济的驱动力。

交易，是经济机器的最基本零件。所有的经济周期和动力，都是交易造成的。

一个人的支出是另一个人的收入，所以你花的越多，别人就挣的越多。

# 央行

央行通过影响利率和发行更多货币来影响经济运行。

# 信贷

信贷是经济中最重要的组成部分。

生产率在长期内最关键，但信贷在短期内最重要。因为生产率在短期内不会剧烈波动。

借贷可以同时满足贷款人（期望钱能变得更多）和借款人（期望消费/开办工厂）的需要。

贷款人和借款人与在交易中的卖方与买房没有两样。

利率高时，借贷变得困难；利率低时，借贷变得容易。

信用良好的借款人具备两个条件：

  • 偿还能力
  • 抵押物

借钱，实质上是向未来的自己借钱。

信贷触发了一系列机械地和可以预料的将在未来发生事件。这就是信贷不同于货币的地方。

现实生活中，大部分所谓的钱实际上是信贷。美国国内的信贷总额大约为50万亿美元，而货币总额只有大约3万亿美元。

# 短期债务周期

信贷可以即刻凭空产生。

一个人的支出时一个人的收入

如果信贷易于获得，经济就会扩张；如果信贷不易获得，经济就会衰退。

# 长期债务周期

总的债务不断增加，因为人们仅注意最近出现的情况，最近的情况是什么呢？收入一直在增加，资产价值不断上升，股票市场欣欣向荣，现在是繁荣时期。

当人们过度借贷消费时，泡沫便产生了，人们感觉自己很富有。

去杠杆（减少长期债务）：

  • 股票市场暴跌，不动产市场一蹶不振，资产价格下跌，银行陷入困境，借款人能够提供的抵押物的价值下降，这进一步降低了借款人的信用。人们觉得自己很穷。
  • 削减支出-->收入减少-->财富减少-->信贷减少-->减少借债-->削减支出....这是一个恶性循环。它看起来与衰退相似，但不同的是，它无法通过降低利率来挽回局面。

# 去杠杆

那么应该怎样应对去杠杆化？问题在于债务负担过重，必须减轻，为此可以采用四种办法。

  • 削减支出：个人、企业和政府削减支出
  • 减少债务：通过债务违约和重组来减少债务
  • 财富再分配：从富人转给穷人
  • 发行货币：央行发行更多货币

萧条的一个主要特征就是人们发现他们原来以为属于自己的财富中有很大一部分实际并不存在。

央行通过新发行的货币，来购买金融资产，帮助推升了资产价格，从而提高了人们的信用。但是这仅仅有助于那些拥有金融资产的人。

中央政府可以通过购买商品和服务，可以向人民送钱，但是无法印钞票。

因此，央行和政府必须合作。

# 经验法则

  • 不要让债务的增长速度超过收入。因为债务负担最终会把你压垮。
  • 不要让收入的增长速度超过生产率，因为这最终将使你失去竞争力。（拿钱不干事，不可持续）
  • 尽一切努力提高生产率，因为生产率在长期内起着最关键的作用。

# 其他

一个善于创新和勤奋的人，将比那些自满和懒惰的人，更快地提高生产率和生活水平，但在短期内不一定能体现出来，生产率在长期内最关键。

每一块钱无论支付的是货币还是信用，对价格的影响都是一样的。
