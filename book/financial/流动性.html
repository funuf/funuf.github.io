# 概述

在经济中，流动性是一个比较重要，但却被很多人忽略的概念。在货币中，流动性体现在，货币从一个人手中流入另一个人手中的速度，或者说货币从银行流动到贷款人手里的速度，这个流动速度大，说明市场上发生的商品买卖活跃，经济就好，但流动性越大被信贷派生出的货币也就越多，有可能会催生出更高的通胀。所以，央行的大部分政策，其实就是在通过控制流动性来控制经济。在资产中，流动性是指，一个资产能被交易的难易程度。资产流动性越好，越容易交易，风险就越小，但价格起伏越小。资产流动性越小，越难交易，价格可能起伏也很大，所以风险就大。所以，在经济不好的时候，投资要首先考虑流动性，因为经济不好，很多资产价格波动就大，如果去投了流动性差的资产，遇到大跌就很难出售，但去投资那些流动性高的东西。因为很容易买进卖出，风险就可控。

# 扫除盲点

央行为何不直接说为了保证市场上货币的充足呢？

其实，流动性这个词还真就比货币这个词更能清晰地表明央行的意图。

钱只有在市场里流动不断交易的时候，才能体现它的价值。把钱放着不动，就是一堆废纸。

假设你有一个亿，但你却把他压在床底下不花出去，对于央行和市场来说，这钱其实可以等同为不存在。

如果人人都不舍得花钱，那发行的货币再多也很难在市场中流动起来。钱不流动，就表示没有交易和买卖发生，经济就没有活力。

这时，央行就会想，通过什么方法才能让钱流动起来呢？让缺钱的人手里有钱，让有钱的人更舍得花钱。

没错，就是信贷。所以，央行说提供信贷支持，其实就是在提供流动性支持。

现代金融体系下，央行早就不单纯印钞了，而是通过信贷的方式去创造货币。信贷发生的越多越频繁，被创造出来的货币就越多。

央行说提供流动性，其实就是让钱从银行到贷款人手里的这个流动速度变快。

那么什么情况下，流动速度会变快呢？也就是贷款人更愿意贷款呢？那当然是利息低的时候。

既然我们已经知道了，信贷是产生货币的重要途径，那么流动性越大，就代表信贷发生的越频繁，市场上被信贷派生出来的货币也就越多。

所以，把投放流动性理解成投放货币也没错。但如果直接说投放货币，会给人造成错觉说央行在直接印钞。

央行没有在直接印钞，只是在降息降准，所以，央行用投放流动性这个词会更准确表达意思。

流动性过剩，就是货币过剩；流动性紧张，就是货币紧张。

对于经济来说，流动性高了或低了都不好。

流动性高了，热钱就会过多，就会引发通胀，以及引发不理性的投资造成资产价格泡沫。

流动性低了，会造成资金成本过高，资产价格下跌，市场缺乏获利，对经济造成负面影响。

所以，刚需只买一套，首先要考虑的还是自己的还贷能力，买能力范围内的房，毕竟房子涨价舍不得卖了变现，这涨价就是虚的，但每一天的还贷压力却是实实在在的。

在整体上，股市的流动性是指股市里资金的活跃程度，在牛市的时候，股市资金的流动性高，成交活跃；在熊市的时候，股市资金流动性低，成交低迷。

很少有庄家坐庄去炒股市里的大盘股，因为大盘股的流动性很高，同一时间买盘和卖盘都很多。如果资金量少，你想买多少就有多少，很难影响价格。
