          # 分类

          人或事，

          分类即是火眼金睛，

          分类是学习、成长、成功的前提，

          分类是事务的着手点，

          分类是处理问题的好的开始。

          # 羊、狗、狼

          # 讨好型人格

          弱势文化熏陶下的差等生，最终会走向取悦身边人的邪路。

          他们会在意别人的眼光与评价，他们会委屈自己以成全别人。期望别人能够肯定自己、接受自己，甚至赞美自己。

          总是被周围的人称为“好人”，他们也乐于听到周围的人称他们为“好人”。

          他们大都逆来顺受：大多的意见都会烂在心里；少数会神神叨叨自言自语。

          殊不知，他们理解错了。

          好人指的是能把事情做好的人，是能把为人处事各方面的事情都做好的人。

          # 狗吃屎狼吃肉

          弱势文化熏陶下的优等生，最终会走向依附救主的邪路。

          他会委屈自己，以求救主的全。期望救主对他网开一面、怜悯，甚至施舍。

          优等生的目标就是成为狗，最好是能吃到被施舍的骨头。当然了，有屎吃他们也是开心的。

          狼吃肉，狗吃屎。巴结别人，摇尾巴的狗总是遭人厌弃的。

          # 单纯的羊

          羊没有去攀附强者，它们表现出顺从仅仅是为了好好表现给强者，以求强者的不杀之恩。

          羊觉得周围的人都是羊，看不出周围的存在的狗，本质上还是阅历太少太嫩了，只看到了高耸的强者和自认为是同伴的狼。

          单纯且不知悔改的羊，前路只有两条：一是被狗弄死，二是被狼弄死。

          # 羊的特点

          羊有一个特点，在通讯录中常常会见到毛主席、周总理、聂帅、小平、彪，等各种称呼，单单不见对方的全名；

          羊觉得这样显得比较亲，比较尊重对方，殊不知自己慢慢地、慢慢地，就这样跪下了。

          # 羊的生存

          羊是在弱势文化熏陶下的长大的，任人宰割却无能为力，只能忍气吞声。

          羊的后续生存有两条路，一是继续卖乖，二是进化为狼。

          # 狼

          狼，外界敏感型，吃肉不靠巴结，会独行，可协作，强势文化的代表。

          狼，有耐心，有能力，冷静。

          狼，敢做困难事。

          狼，协调得了鳄鱼和猴子。

          狼，不搞个人崇拜，离开狼王，狼依然能生存。

          狼，不会对牛弹琴。

          狼，知道周围的人形形色色，看人十二属。

          狼知道不是所有的人都能变成狼，

          狼除了引导他的孩子走向成狼之路，不会期望让其他人变成狼。

          # 十二属

          初识：兔

          上行：羊、牛、鸡、马、虎

          下行：猴、猪、鼠、龙、狗、蛇

          如果没有更多的接触、更深的了解，则属相止步。

          切不可因为认识时间长，就拿兔当马，这个兔很可能是个狗。

          1. 一个人往往是十二属的综合体，多多少少都会占一方面，但有主次之分
          2. 人是活的人，是变化着的人，要以开放的、动态的眼光去审视所属是否发生了变化
          3. 江山易改，本性难移，虽然能改，但很少

          # 鼠

          偷心太重，投机取巧，好高骛远

          # 牛

          踏实肯干，吃苦耐劳，具有力量

          # 虎

          真实的强者，百兽之王，可遇不可求

          你我皆成为虎，难。

          你我成为小虎——狼，可期。

          # 兔

          无害无益

          家兔胆小

          野兔敏捷

          # 龙

          虚构的强者，自以为是，亢龙有悔

          # 蛇

          冷血不孝顺，阴险、狠毒

          # 马

          有能力，有速度，人腿的延伸，效率的保证

          真的忠诚：忠于事业，忠于信仰

          # 羊

          善群，流于大众，没有主见

          好仁，善良，有角但不好斗

          死义，被宰杀时安静，任人宰割

          知礼知恩，跪乳象征孝道

          # 猴

          无臀，坐不住，不稳当

          急性子、火急火燎

          机敏

          好学，但学的是样子

          # 鸡

          头戴冠，文也；足搏距，武也；见敌敢干，勇也；见食相呼，义也；守夜不失时，信也

          奈何实力太弱，寄人篱下

          # 狗

          本狗忠诚

          人狗吃屎

          人无需学狗

          忠诚于信仰、忠诚于事业才是真的忠诚

          忠诚于上级、忠诚于个人只是企图吃屎

          # 猪

          主要是懒，好吃懒作，妄图天上掉馅儿饼

          惰性太强

          目光短浅

          # 论担当

          有担当，不是埋头苦干，只把自己的事做好，而是要把需要自己处理的所有事情都处理好。

          有担当的人能够及时响应外界发出的信号，而不是自我沉浸在自己想象的美好中独自安好。

          # 具备敏感型

          对事，要具备敏感性。

          听到了家人/朋友/同事/陌生人的吩咐、交代、嘱托，要及时反应。不能总觉得自己的事是最重要的，对他们的话就当作耳旁风，不放在心上。

          但也不是他们说的所有事都无条件/立刻执行。分析充分性和必要性，该现在做的就立即做雷厉风行，需要稍等的就稍等做总不心急，没必要做的就不做毫不迁就。

          比学习更重要的是生活，因为学习的目的就是更好的生活；

          比工作更重要的是家人，因为工作的目的就是给家人保障；

          比理论更重要的是实践，因为理论的目的就是要指导实践。

          其实也不能有xxx比yyy更重要的想法，需要当下自己做好的都是重要的，都值得耐心去做。

          # 不怕事

          事，就是矛盾。矛盾永远会存在，事情也永远都会发生。它们不会因为你怕就自动消失。

          对事，需要的是面对、分析、执行亦或接受。

          # 担当的表现

          对于那些虽有困难，但充分且必要的事，必须做，不要怕。

          对于那些解决不了的事情，任他去吧，不要放在心上。

          对于那些现阶段解决改变不了/不能解决的事情，不急躁，不强求，给时间，做好准备，任他发展。

          对于那些有能力/经常做的事情，耐心做好不毛躁，不轻视它，要总结经验提高效率，量的积累才会有质的飞跃。

          # 我的盲点

          • 要把所有的事情都处理好，而不是把是所有的事情都做好。
          • 好人，不是对别人老好的人，而是能把事情处理好的人。
          • 讨好若有用，还要实力干什么。
          • 弱肉强食：真理/法律/规则的适用，只在大炮（实力）的射程范围之内。

          # 狼的境界

          1. 尽人事，听天命。君子务本，本立而道生。
          2. 是道则进，非道则退。
          3. 应无所住而生其心。
